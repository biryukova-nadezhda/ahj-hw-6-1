Класс Theme предназначен 

Структура класса:
1. constructor()
2. init()
3. firstInit()
4. changeTheme(event)
5. getActiveTheme()
6. applyTheme(themeName)

Краткое описание методов:
1. constructor() - происходит привязка кнопок смены темы и ссылки на css файл

2. init() - Метод назначает обработчики событий клика на кнопки смены темы

3. firstInit() - устанавливает тему по умолчанию при первом запуске сайта

4. changeTheme(event) - Метод производит проверку, если нажатая кнопка содержит класс активе, то это используемая тема и ничего делать не нужно, если не нет, то удаляем с других кнопок класс active, назначаем нажатой и меняем адрес на css-файл по типу кнопки

4. getActiveTheme() - метод для работы с localStorage. Выдает строку с названием темы

5. applyTheme(themeName) - принимает строку с названием темы и меняет ссылку на файл со стилями, а также добавляет класс active кнопке с активной темой
